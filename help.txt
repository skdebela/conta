Conta Contact Manager - Usage

Commands:
  -A, --add           Add a new contact
    -n, --name        Name of the contact
    -p, --phones      Comma-separated phone numbers
    -e, --emails      Comma-separated emails
    -c, --categories  Categories (comma-separated)

  -S, --search        Search for contacts
  -C, --categorize    View contacts by category

  -h, --help          Show this manual

Run `conta` without arguments for an interactive menu.
